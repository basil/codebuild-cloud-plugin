AWS credentials for accessing AWS CodeBuild.

If you configured an IAM role, ensure that it includes the following policy statements (where {PROJECTNAME} is replaced with the name of your project):

[
  {
    "Effect": "Allow",
    "Resource": "*",
    "Action": ["codebuild:ListProjects"]
  }
  {
    "Effect": "Allow",
    "Resource": "arn:aws:codebuild:*:*:project/{PROJECTNAME}",
    "Action": ["codebuild:StartBuild", "codebuild:StopBuild"]
  }
]
